# Jimmy Liou

資深前端主任工程師／前端技術管理 & 架構設計

從後端工程師起步，歷經電商、旅遊、HR SaaS 等產業，逐步走向前端架構設計與技術管理。一路以來最在意的事情沒變過：**怎麼用工程手段讓團隊少做重複的事**。目前在 MAYO 鼎恒數位科技負責前端技術方向，專注在 Monorepo 架構、微前端、以及 AI 輔助開發流程的落地。

---

### 前端技術經理 | 主任工程師 — MAYO 鼎恒數位科技

**2021.05 – Present**

帶領前端團隊，從零建構 Nx Monorepo + 微前端架構，整合多個前端應用。同時推動 Storybook + React + Tailwind 標準化元件庫，讓跨團隊能共用同一套設計語言。在 DevOps 面向，設計了 Azure DevOps Pipeline + Docker/K8s (AKS) 的部署流程。

幾個比較有感的成果：

| 指標 | 改善幅度 | 怎麼做到的 |
|------|----------|-----------|
| 元件重用率 | **+40%** | 標準化 UI 元件庫，跨團隊共用 |
| 開發時程 | **-33%** | 3 天 UI 任務縮短至 2 天內 |
| Build 時間 | **8→2 分鐘** | Nx Cache + 分散式快取優化 |
| 前端人力需求 | **-50%** | 架構標準化後，同樣產出需要更少人力 |

其中最有趣的項目是 **AI-Driven PR Reviewer Agent** — 用 OpenAI GPT + Function Calling 打造的智能代碼審查系統，自動分析 PR Diff 與 Commit 歷史，生成審查建議與風險提示，整合進 CI/CD 流程做到 PR 建立即觸發。

### 前端高級工程師 — 博裕科技（2017–2021）

這段時期是從「寫功能」轉向「建體系」的階段。用 Node.js + Next.js (SSR) 搭建全端架構，導入 GraphQL 減少 30% 冗餘 API 請求，透過代碼分割將 Bundle Size 縮減 35%。同時建立了完整的品質保證流程 — ESLint/TypeScript/Prettier 規範、200+ 測試案例、AWS ECS 容器化部署與零停機佈署。重大線上事故從每月 2 次降到幾乎為零。

### 後端工程師 — 亞卡西雅 & 喜美旅行社（2011–2017）

職涯起點。在電商與旅遊產業做全端開發，負責前台網站建置、後台系統維護、活動頁面製作。這幾年打下了對 Web 開發的基本功，也讓我開始意識到前端體驗對產品的影響力，最終決定往前端深耕。
